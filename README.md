## ğŸ”§ Installation & Setup

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Git (optional)

### Local Development

1. **Clone repository**
   ```bash
   git clone https://github.com/PieByter/my-portfolio.git
   ```

2. **Navigate to project**
   ```bash
   cd my-portfolio
   ```

3. **Open in browser**
   - Buka `index.html` di browser
   - Atau gunakan Live Server extension di VS Code

## âœï¸ Customization Guide

### 1. Personal Information
Edit di `index.html`:
- Ganti "Your Name" dengan nama Anda
- Update bio di section About
- Ganti email, phone, location di section Contact

### 2. Projects
Edit di `index.html` section Projects:
- Update project titles
- Ganti descriptions
- Update technology tags
- Tambahkan link GitHub & Live Demo Anda

### 3. Skills
Edit di `index.html` section Skills:
- Update skill names
- Adjust skill levels (data-progress)
- Tambah/hapus skills sesuai keahlian

### 4. Color Scheme
Edit di `style.css` `:root` variables:
```css
--color-accent-primary: hsl(200, 85%, 60%);    /* Sky Blue */
--color-accent-secondary: hsl(210, 90%, 55%);  /* Bright Blue */
--color-accent-tertiary: hsl(220, 80%, 50%);   /* Ocean Blue */
```

### 5. Social Media Links
Edit di `index.html` section Contact:
- Update GitHub, LinkedIn, Instagram, Twitter links

## ğŸŒ Deployment

### Deploy ke GitHub Pages

1. **Push ke repository**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Buka Settings â†’ Pages
   - Source: Deploy from branch `main`
   - Folder: `/ (root)`
   - Save

3. **Akses website**
   - URL: `https://piebyter.github.io/my-portfolio`
   - Tunggu 1-2 menit untuk build

### Deploy ke Netlify (Alternative)

1. Drag & drop folder ke [Netlify Drop](https://app.netlify.com/drop)
2. Atau connect GitHub repository untuk auto-deploy

1. Import project ke [Vercel](https://vercel.com)
2. Auto-deploy setiap kali push ke GitHub
